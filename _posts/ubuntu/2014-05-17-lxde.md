# LXDE

Ubuntu 12.04 server + LXDE

## 0. Install

    sudo apt-get install lxde
    sudo apt-get install xinit mdm

## Font

    sudo apt-get install ttf-arphic-uming

## Remote Desktop Client
for connect to win7 RDP
