# Squid 3
proxy server

## Install

    sudo apt-get install squid3 apache2-utils

## Config
/etc/squid3/squid.conf

## Authentication
Add User

    htpasswd /etc/squid3/passwd user1


## Startup

    sudo service squid3 start
