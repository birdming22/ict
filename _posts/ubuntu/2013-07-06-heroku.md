# Heroku

## Ubuntu Client
ref. [heroku toolbelt](https://toolbelt.herokuapp.com/debian)

    wget -qO- https://toolbelt.heroku.com/install-ubuntu.sh | sh

## Getting Started

    heruko login
    cd project
    heruko create
    git push heruko master
