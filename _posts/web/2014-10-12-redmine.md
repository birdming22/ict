# Redmine
Only for [Bitnami Redmine](https://bitnami.com/stack/redmine)

## Backup

    cd redmine-2.3.0-0
    ./ctlscript.sh stop
    ./ctlscript.sh start mysql
    mysql -u root -p -h 127.0.0.1
    ./mysql/bin/mysqldump -u root -p --all-databases > redmine.sql
    
## Upgrade
backup before upgrade
