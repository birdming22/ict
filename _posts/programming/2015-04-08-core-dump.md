# Core Dump

## Example

file: example.c

    #include <stdlib.h>
                                                                     
    int main()
    {                                                                   
        abort();
    }

setting

    ulimit -a
    ulimit -c unlimited

compile and run

    gcc -g -o example example.c
    ./example
    Aborted (core dumped)

core file

    gdb ./example core
