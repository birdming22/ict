# Valgrind

## Install for ubuntu

    sudo apt-get install valgrind
    
## Usage
ref. [Using Valgrind to Find Memory Leaks and Invalid Memory Use](http://www.cprogramming.com/debugging/valgrind.html)

    valgrind --tool=memcheck --leak-check=yes example1
