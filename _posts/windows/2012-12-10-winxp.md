## PuTTY 免密碼登入
ref. [PuTTY 免密碼登入](http://josephj.com/article/putty-key/)

- 下載 private key, ex. id.rsa
- 用 PuTTY Key Generator 產生 PuTTY 專用的 private key，然後 Save private key
- 將原本的 id.rsa 刪除，保留 PuTTY 專用的 private key 就好。
